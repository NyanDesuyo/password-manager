# Password Manager

## Warning

Not ready for Production!!

## About

Secure your account using NodeJS, LowDB, and encrypted

## Instalation

NodeJS v15.11.0

```
npm install
npm run build

1st time run
npm run register
npm run generate

usage
npm start
```

if you're forgot your username and password

```
npm run reveal
```
